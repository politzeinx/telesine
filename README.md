#Telesine
 
Site de catálogo de filmes inspirado no Letterboxd, desenvolvido com HTML e CSS puro.
 
##Sobre o projeto
 
O Telesine é uma página web de catálogo de filmes com visual escuro e moderno. O projeto foi desenvolvido do zero usando apenas HTML e CSS, sem frameworks ou bibliotecas externas.
 
##Funcionalidades
 
- Navbar com logo colorido, menu de navegação e botão de login
- Seções de filmes organizadas em grade
- Banners entre as seções
- Layout totalmente centralizado e responsivo
- Rodapé com links e redes sociais
 
##Tecnologias usadas
 
- HTML5
- CSS3
- Google Fonts (Playfair Display + DM Sans)
- SVG para ícones
 
##Estrutura de arquivos
 
```
telesine/
├── index.html
├── styles.css
├── favicon/
│   └── favicon.png
└── img/
    ├── filme1.jpg
    ├── filme2.jpg
    └── ...
```
 
##Como rodar
 
Basta abrir o arquivo `index.html` no navegador. Não precisa instalar nada.
